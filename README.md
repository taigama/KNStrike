# KNStrike
2D-Android game, Cocos2d-x 3.16

## Target
- 2D game
- Top-down view, vertical screen
- Android 21+

## Technology
- GameEngine: Cocos2d-x 3.16.
- Physics: Box2d.
- Sound: AudioEngine.
- Windows IDE: Visual Studio 2017.
- Build, Debug & Android IDE: Android Studio.
